# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any of my projects, please report it responsibly.

**Do not open a public issue.**

Instead, please email or DM me directly so I can assess and address the issue before it becomes public.

## What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if you have one)

## Response

I'll acknowledge your report within 48 hours and aim to provide a fix or mitigation plan within 7 days, depending on severity.

## Scope

This policy applies to all public repositories under [stussysenik](https://github.com/stussysenik).
